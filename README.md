# fcuk-Keil-MDK-Community-Edition

移除keilμVision5中的社区许可证

## 项目介绍

keil5这个社区许可证犹如狗皮膏药，一般人找不到彻底移除它的方法
我写了一个PowerShell脚本，用于移除Keil μVision5中的社区许可证

## 使用方法

1. 下载 `remove_keil_license.ps1` 脚本文件
2. 右键点击脚本文件，选择"使用PowerShell运行"
3. 脚本会自动请求管理员权限
4. 按照屏幕提示操作即可

## 注意事项

- ⚠️ 本脚本仅供学习和研究使用
- ⚠️ 运行时弹的文件窗口是给你看的，记得返回脚本界面继续下一步
- ⚠️ 如果脚本一闪而过，八成是你在设置里开"允许脚本运行"
![alt text](data/image.png)
- ⚠️ 操作过程中请勿关闭脚本窗口

## 干啥的？

脚本主要执行以下操作：
- 检查当前用户权限
- 定位 `.armlm` 许可证文件夹
- 删除许可证缓存、锁文件、日志等

## 系统要求

- PowerShell 5.0 或更高版本
- 管理员权限
